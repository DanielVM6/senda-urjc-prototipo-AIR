# Prototipo Senda URJC 🚶‍♀️📱

Este repositorio contiene el prototipo evolutivo de la aplicación **Senda URJC**, desarrollado como parte de la práctica de Análisis e Ingeniería de Requisitos. 

El sistema es una solución de movilidad inteligente diseñada para la comunidad universitaria de la URJC, que prioriza el cálculo de "Rutas Seguras" basándose en la iluminación, afluencia y el estado del entorno.

## 🛠️ Tecnologías utilizadas
* **Lenguaje:** Python 3.x
* **Framework Interfaz:** Streamlit

## 🚀 Instrucciones de despliegue y ejecución

Para ejecutar esta aplicación en tu entorno local, sigue estos pasos:

### 1. Requisitos previos
Asegúrate de tener instalado Python en tu sistema. Puedes descargarlo desde [python.org](https://www.python.org/).

### 2. Instalación de dependencias
El proyecto utiliza Streamlit para la interfaz gráfica. Para instalarlo, abre tu terminal o consola de comandos y ejecuta:

`pip install streamlit`

### 3. Ejecución de la aplicación
Una vez instalada la librería, navega en tu terminal hasta la carpeta donde hayas descargado el archivo `main.py` y ejecuta el siguiente comando:

`streamlit run (nombre fichero ej:"main.py")`

Esto abrirá automáticamente una nueva pestaña en tu navegador web predeterminado (típicamente en `http://localhost:8501`) con la aplicación funcionando.
